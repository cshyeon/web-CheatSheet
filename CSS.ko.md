### <a id="p-index"></a>p
- pseudo class selector
```css
:link - 방문한 적이 없는 링크
:visited - 방문한 적이 있는 링크
:hover - 마우스를 롤오버 했을 때 
:active - 마우스를 클릭하여 누르고 있을 때
:focus - 해당 엘레먼트가 포커스 상태일 경우
```
